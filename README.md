A simple calculator GUI application for the video game ROUNDS by Landfall Games. If you play the game competitively, it might be useful to know how specific cards impact your overall damage. This application mainly provides information about:

- **Damage per Bullet**
- **Maximum bullet DPS**
- **Total damage per block**
- **Maximum block DPS**
- **Total health indicator**
- **Reload time**
- **Attack speed**


## Installation
Clone this repository or download the ZIP file. Inside the folder, open a command prompt and run:

```bash
pip install -r requirements.txt
```


Disclaimer: Many of the damage values were found through rigorous in-game testing. Because of that, actual damage values may not exactly coincide with in-game experience. Keep that in mind while using the program!
